# local start

## git clone
このプロジェクトをクローンしてください。
## node install
nodeはnvm推奨(権限エラー回避の為)

```
npm install hexo-cli -g
npm install

```

## 起動

```
hexo s --debug
hexo clean && hexo deploy --generate

```

http://localhost:4000 で起動するか確認する

# congfigに関して
configはHexoとNextのものを編集することになります。
